# **App Name**: Sukasuka Setudio Landing

## Core Features:

- Hero Section: Display the studio name, tagline (Raya dengan suka suka), a minimal CTA button, and a reserved space for a hero image.
- About Section: Showcase a short and refined description of the creative studio.
- Services Section: Present the studio's services in a simple grid layout, with optional icons.
- Portfolio Gallery: Display a gallery or portfolio of the studio's work, with clearly marked image placeholders.
- Contact Section: Include a contact section with a minimal form or contact information.

## Style Guidelines:

- Primary color: Black (#000000) for a high-end, minimalist aesthetic.
- Secondary colors: Light gray tones (#D3D3D3) to complement the black.
- Accent: Subtle silver gradients or thin silver lines (#C0C0C0) for a premium touch.
- Background: Mostly black or off-black (#121212), with high-contrast text.
- Font: 'Inter' sans-serif font, modern style. Note: currently only Google Fonts are supported.
- Large bold headlines and thin, elegant subtext for a premium feel.
- Letter-spacing slightly increased for a sophisticated appearance.
- No clutter, emphasizing whitespace for a minimalist design.
- Soft hover effects only and smooth transitions for a refined user experience.
